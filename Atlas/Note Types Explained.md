# Note Types Explained

## Quick Reference

| Type | Template | Where It Lives | Lifespan | Purpose |
|------|----------|---------------|----------|---------|
| Fleeting Note | `Fleeting Note` | `00 - Inbox/` | < 48 hours | Quick capture |
| Literature Note | `Literature Note` | `Sources/` | Permanent | Summarize a source |
| AI Paper Review | `AI Paper Review` | `Sources/` | Permanent | Deep paper analysis |
| Permanent Note | `Permanent Note` | `Atlas/` | Forever | Atomic, original idea |
| Project Note | `Project Note` | `01 - Projects/` | Until done | Track active work |
| MOC | `MOC Template` | `Maps/` | Evolving | Topic hub / index |
| Daily Note | `Daily Note` | Root or dedicated folder | 1 day | Daily planning |
| Weekly Review | `Weekly Review` | Root or dedicated folder | 1 week | GTD review ritual |
| Prompt Template | `Prompt Template` | `03 - Resources/` or `Atlas/` | Permanent | Reusable AI prompts |
| Tool Review | `Tool Review` | `Sources/` or `03 - Resources/` | Permanent | Evaluate AI tools |
| Meeting Note | `Meeting Note` | `00 - Inbox/` then file | Process quickly | Meeting capture |
| Experiment Log | `Experiment Log` | `01 - Projects/` | Project lifetime | Track AI experiments |

## When to Use What

- **Just had a random thought?** -> Fleeting Note
- **Reading/watching something?** -> Literature Note
- **Reading a research paper?** -> AI Paper Review
- **Had an original insight?** -> Permanent Note
- **Starting something with a goal?** -> Project Note
- **Need to organize a topic?** -> MOC Template
- **Planning your day?** -> Daily Note
- **End of week?** -> Weekly Review
- **Found a great prompt?** -> Prompt Template
- **Trying out an AI tool?** -> Tool Review
- **In a meeting?** -> Meeting Note
- **Running an AI experiment?** -> Experiment Log
