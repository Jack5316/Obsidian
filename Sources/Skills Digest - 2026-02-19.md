# AI Agent Skills Digest

A curated list of the most valuable skills for developers using Claude Code and Obsidian, organized for quick reference and implementation.

---

## Top Picks

### 1. find-skills
The ultimate discovery tool—lets you search and explore the entire skills.sh directory directly from your agent. Essential for building your skill library.  
`npx skills add vercel-labs/skills`

### 2. vercel-react-best-practices
Official Vercel guidance for building scalable, performant React apps. Perfect for leveling up your component architecture and avoiding common pitfalls.  
`npx skills add vercel-labs/agent-skills`

### 3. react-doctor
Diagnoses and fixes common React performance issues, anti-patterns, and bugs. A must-have for maintaining healthy React codebases.  
`npx skills add millionco/react-doctor`

### 4. next-best-practices
Official Next.js best practices from Vercel—covers App Router, data fetching, deployment, and optimization for production apps.  
`npx skills add vercel-labs/next-skills`

### 5. supabase-postgres-best-practices
Expert guidance for Supabase and Postgres—schema design, security (RLS), performance tuning, and real-time features.  
`npx skills add supabase/agent-skills`

### 6. browser-use
Advanced browser automation for testing, scraping, and interactive workflows. Integrates seamlessly with AI agents for complex web tasks.  
`npx skills add browser-use/browser-use`

### 7. skill-creator
Create your own custom Claude Code skills. Essential for tailoring your agent to specific project needs and team workflows.  
`npx skills add anthropics/skills`

### 8. brainstorming
Structured brainstorming workflows for ideation, problem-solving, and project planning. Great for kickstarting new features or debugging sessions.  
`npx skills add obra/superpowers`

### 9. systematic-debugging
A methodical approach to debugging—helps you isolate issues, form hypotheses, and validate fixes efficiently. Reduces time spent chasing bugs.  
`npx skills add obra/superpowers`

### 10. test-driven-development (TDD)
Guides you through the TDD workflow—red, green, refactor—with practical examples for React, Next.js, and Python. Improves code quality and confidence.  
`npx skills add obra/superpowers`

### 11. ui-ux-pro-max
Comprehensive UI/UX guidance for modern web apps—covers design systems, accessibility, responsive design, and user testing.  
`npx skills add nextlevelbuilder/ui-ux-pro-max-skill`

### 12. pdf
Generate, manipulate, and extract content from PDFs directly in your agent workflows. Useful for reports, documentation, and data processing.  
`npx skills add anthropics/skills`

### 13. webapp-testing
End-to-end and unit testing guidance for web apps—covers tools, strategies, and best practices for reliable test suites.  
`npx skills add anthropics/skills`

### 14. better-auth-best-practices
Expert guidance for implementing secure authentication—covers session management, OAuth, password security, and common auth pitfalls.  
`npx skills add better-auth/skills`
