# Weekly Synthesis Skill

Creates cross-source synthesis of all content generated during the week.

## Usage

```bash
python3 _scripts/weekly_synthesis.py
```

## Output

`Sources/Weekly Synthesis - YYYY-MM-DD.md` (themes across ArXiv, HN, Reddit, news, Twitter)
